# 模块一：确定性的回归 (Determinism & High-Dimensionality)

### 1. 理论假说：维度缺失产生的“概率假象”
在传统 3 维物理观测中，微观粒子表现出随机跳跃和不确定性。本模块旨在通过数学演示证明：所谓的“随机”，仅仅是高维有序运动在低维空间的投影。

当我们将系统放入 5 维几何流形（由 $r$ 轴和相位 $\phi$ 锁定）时，粒子的路径是绝对确定的本征曲线。

### 2. 实验：v1-概率预测模拟
通过 `v1_probability.py`，我们模拟了一个受 Twin-G 屏障锁定的 5 维粒子。代码展示了当我们将 5 维确定性轨道投影回 3 维，并引入观测偏差时，它是如何呈现出“概率云”特征的。

**结论**：不确定性是观测能力的缺失，而非宇宙的本质。
