# Twin-G Framework (AtomGrid Core)

Twin-G 是一种基于 **5 维几何流形** 的自洽性理论框架，旨在解决多主体系统中的混沌冲突与物理碰撞问题。

## 核心原理
本项目通过引入高阶几何排斥项，模拟微观原子级的稳定锁定：

$$a = \frac{G}{dist^2} - \frac{\kappa}{dist^5}$$

- **引力项 ($1/r^2$)**: 维持系统的宏观联系与向心力。
- **屏障项 ($1/r^5$)**: 在近距离产生极强的几何刚性，确保实体永不碰撞。

## 本征常数 (Captured Constants)
在模拟中，我们发现了确保系统进入拓扑稳态的共振数值：
- **相位步长**: `0.3065`
- **几何刚性缩放**: `1.37`
