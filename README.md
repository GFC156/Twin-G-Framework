# Twin-G Framework (AtomGrid Core)

> **"在高维几何的秩序下，不确定性只是低维观测的幻觉。"**

Twin-G 是一种基于 **5 维几何流形 (5D Manifold)** 的自洽性理论框架。它旨在通过引入 $1/r^5$ 的高阶几何屏障力，从几何角度统一解释微观粒子的确定性路径与能级塌陷现象。

---

## 核心物理图景 (The Vision)

在传统的 3 维空间中，微观粒子表现出“概率云”和“不确定性”。Twin-G 模型假设，这种随机性源于 **观测维度的缺失**。

本项目通过引入一个与三维空间正交的第五维度（$r$ 轴），构建了一个受高阶几何屏障保护的流形。在这种模型下，粒子不再是随机跳跃的，而是被“锁”在极其稳定的几何沟壑中。

### 核心动力学方程 (Core Equation)
我们定义了实体的相互作用力，它由经典引力项与高阶几何排斥项组成：

$$a = \frac{G}{dist^2} - \frac{\kappa}{dist^5}$$

- **引力项 ($1/r^2$)**: 维持系统的宏观联系与向心力。
- **屏障项 ($1/r^5$)**: 5 维几何的本征刚性。在近距离产生极强的几何排斥，确保实体永不碰撞。

### 本征常数 (Captured Constants)
在模拟过程中捕捉到的共振数值：
- **相位步长 ($\phi$): `0.3065`**
- **几何刚性缩放 ($\kappa$): `1.37`**

## 深度研究模块 (Research Modules)

本项目采用模块化设计，目前已完成以下核心研究验证：

### 01. [确定性的回归 (Determinism)](./modules/01_determinism/)
- **核心实验**: `v1_probability.py`
- **内容**: 证明概率分布本质上是几何路径的时间权重投影。通过算法展示 5 维本征轨道如何由于维度坍缩，在 3 维观测中伪装成“概率云”。

### 02. [氢原子能级与光谱阶梯 (Hydrogen)](./modules/02_hydrogen/)
- **核心实验**: `v2_hydrogen.py` (v9.0 架构)
- **内容**: 基于 5 维驻波条件，模型自发演化出了基态能级为 **$-13.60$ eV** 的能量阶梯，完美匹配 **Lyman-alpha** 谱线。
